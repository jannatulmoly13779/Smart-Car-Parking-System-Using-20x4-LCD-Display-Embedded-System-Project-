# 🚗 Smart Car Parking System Using 20x4 LCD Display

## 📌 Project Overview

This project is an **embedded system-based smart car parking system** that displays real-time parking slot availability using a **20x4 LCD display**. It helps manage parking efficiently by tracking entry and exit of vehicles.

The system is designed to reduce manual effort and improve parking space utilization.

---

## 🎯 Features

* Real-time parking slot monitoring
* 20x4 LCD display for clear status visualization
* Automatic vehicle detection using sensors
* Entry and exit tracking
* Efficient and low-cost solution

---

## 🛠️ Technologies Used

* Microcontroller (e.g., Arduino / ATmega)
* 20x4 LCD Display
* IR Sensors / Ultrasonic Sensors
* Embedded C / Arduino IDE
* Basic Electronics Components

---

## ⚙️ How It Works

1. Sensors detect vehicle entry and exit.
2. Microcontroller processes the data.
3. Available parking slots are calculated.
4. The LCD display shows:

   * Total slots
   * Available slots
   * Occupied slots

---

## 📷 System Architecture

* Input: Sensors (IR / Ultrasonic)
* Processing: Microcontroller
* Output: 20x4 LCD Display
